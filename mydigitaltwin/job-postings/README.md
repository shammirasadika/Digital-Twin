# Job Posting Template

**Instructions**: 
1. Go to https://www.seek.com.au/
2. Find a job that matches your skills
3. Copy the entire job posting (title, company, description, requirements, etc.)
4. Paste it into a new file: `job1.md`, `job2.md`, etc.

## Example Structure:

```
# [Job Title]

**Company**: [Company Name]
**Location**: [City, State]
**Salary**: [Salary Range]
**Job Type**: [Full-time/Part-time/Contract]

## About the Role
[Job description]

## Key Responsibilities
- Responsibility 1
- Responsibility 2
- Responsibility 3

## Key Selection Criteria / Requirements
- Requirement 1
- Requirement 2
- Requirement 3

## About the Company
[Company information]

## How to Apply
[Application instructions]
```

---

**Note**: Once you add a job posting, you can use GitHub Copilot to simulate an interview based on this job description.
